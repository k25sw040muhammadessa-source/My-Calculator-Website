# Calculator Website

A modern, fully-functional web calculator built with HTML, CSS, and JavaScript. Features a sleek glass-morphism design with a dark theme and smooth animations.

## 🌟 Features

### Core Functionality
- **Basic Arithmetic Operations**: Addition (+), Subtraction (−), Multiplication (×), and Division (÷)
- **Percentage Calculations**: Quick percentage computations
- **Decimal Support**: Full support for decimal numbers
- **Delete Function**: Remove the last digit with the DEL button
- **Clear Function**: Reset the calculator with the AC (All Clear) button
- **Error Handling**: Built-in protection against division by zero

### User Interface
- **Glass Morphism Design**: Modern frosted glass effect with backdrop blur
- **Dark Theme**: Eye-friendly dark color scheme with purple/indigo accents
- **Responsive Layout**: Works seamlessly on desktop and mobile devices
- **Animated Background**: Floating gradient animations for visual appeal
- **Hover Effects**: Smooth transitions and interactive button feedback
- **Display Information**: Shows current calculation and previous operations

### Input Methods
- **Mouse/Touch**: Click buttons to perform calculations
- **Keyboard Support**: Full keyboard integration
  - **Numbers**: Press 0-9 to input numbers
  - **Operators**: Use `+`, `-`, `*`, `/` to select operations
  - **Enter or `=`**: Calculate result
  - **Backspace**: Delete last digit
  - **Escape**: Clear calculator
  - **Decimal**: Press `.` to add decimal point

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No installations or dependencies required

### Installation
1. Clone or download the project files
2. Open `index.html` in your web browser
3. Start calculating!

## 💡 Usage

### Basic Calculations
1. Click on number buttons to input values
2. Select an operator (+, −, ×, ÷)
3. Enter the second number
4. Press the equals button (=) or press Enter on your keyboard to see the result

### Example Calculations
- **Addition**: 5 + 3 = 8
- **Percentage**: 200% = 2 (or 50 of 200 = 100)
- **Decimal**: 10.5 × 2 = 21
- **Division**: 100 ÷ 4 = 25

## 🔧 Technical Details

### JavaScript Architecture
- **Object-Oriented Design**: Calculator implemented as a class
- **Event-Driven**: Listens to both mouse clicks and keyboard input
- **Floating-Point Precision**: Handles decimal calculations with rounding to prevent floating-point errors
- **State Management**: Maintains display state, current input, previous input, and operator

### CSS Features
- **CSS Variables**: Organized color and spacing variables for easy customization
- **Backdrop Filter**: Glass-morphism effect using CSS blur
- **Flexbox**: Responsive button grid layout
- **Animations**: Keyframe animations for background floating effect

### Browser Compatibility
- Modern browsers with CSS Grid and Flexbox support
- Backdrop filter support for best visual effects
- Tested on: Chrome, Firefox, Safari, Edge
